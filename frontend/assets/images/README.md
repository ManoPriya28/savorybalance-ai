# Image Assets for SavoryBalance AI

## Recommended Images to Add:

### 1. Logo (logo.png)
- Dimensions: 200x200px
- Format: PNG with transparent background
- Style: Green/teal gradient with leaf icon

### 2. Food Plate (plate.png)
- Dimensions: 400x400px
- Format: PNG
- Content: Circular plate with food sections

### 3. Food Icons (food-icons/)
Create a folder with individual food icons:
- chicken.png
- broccoli.png
- avocado.png
- yogurt.png
- etc.

### 4. Favicon (favicon.ico)
- Dimensions: 16x16, 32x32, 48x48
- Content: Simple leaf icon

## If you don't have images, you can:

### Option A: Use Font Awesome Icons (Already in use)
We're already using Font Awesome icons throughout the app.

### Option B: Download free icons from:
1. **Flaticon** (free with attribution)
2. **Font Awesome** (free icons)
3. **Icons8** (free with attribution)

### Option C: Generate with AI:
Use AI image generators to create:
- Nutrition-themed backgrounds
- Food illustrations
- Gradient patterns

## Placeholder Implementation:

If you don't add images, the app will use:
1. Font Awesome icons (already included)
2. CSS gradients for visual appeal
3. SVG shapes for decorative elements

The app is fully functional without custom images!
